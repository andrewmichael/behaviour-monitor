<feature>
  <name>RoutineModel — Per-Entity Baseline Learning Engine</name>
  <files>custom_components/behaviour_monitor/routine_model.py, tests/test_routine_model.py</files>
  <behavior>
    ## ActivitySlot
    - slot_index(hour=14, dow=2) returns 2*24+14 = 62
    - slot_index(hour=0, dow=0) returns 0; slot_index(hour=23, dow=6) returns 167
    - New slot has is_sufficient=False
    - After 4+ event_times recorded, is_sufficient=True
    - Binary: recording "on" at timestamp adds to event_times deque (maxlen=56)
    - Binary: expected_gap_seconds returns median of inter-event intervals when sufficient, None when insufficient
    - Numeric: recording "21.5" updates Welford accumulators (numeric_mean, numeric_m2, numeric_count)
    - Numeric: slot_distribution returns (mean, stdev) tuple when sufficient, None when insufficient
    - to_dict/from_dict round-trips preserve all data including deque contents and Welford accumulators

    ## EntityRoutine
    - record(timestamp, state_value) routes to correct slot via hour/dow
    - Binary entity with "on"/"off"/"open"/"closed" states records event times
    - Numeric entity with "21.5" string state records parsed float into Welford
    - confidence() returns 0.0 with no data
    - confidence() returns ~0.5 when first_observation is 14 days ago with 28-day window
    - confidence() returns 1.0 when first_observation is 28+ days ago with 28-day window
    - daily_activity_rate(date) counts state changes on that date
    - to_dict/from_dict round-trips preserve entity_id, is_binary, all 168 slots, first_observation

    ## RoutineModel
    - get_or_create(entity_id, is_binary) creates new EntityRoutine on first call, returns same on second
    - record(entity_id, timestamp, state_value, is_binary) delegates to correct EntityRoutine
    - overall_confidence() returns 0.0 with no entities
    - overall_confidence() returns average of all entity confidences
    - learning_status() returns "inactive" when overall_confidence < 0.1
    - learning_status() returns "learning" when 0.1 <= overall_confidence < 0.8
    - learning_status() returns "ready" when overall_confidence >= 0.8
    - to_dict/from_dict round-trips preserve all entities and their routines

    ## Entity Type Detection (helper for coordinator)
    - is_binary_state("on") returns True
    - is_binary_state("off") returns True
    - is_binary_state("open") returns True
    - is_binary_state("closed") returns True
    - is_binary_state("locked") returns True
    - is_binary_state("unlocked") returns True
    - is_binary_state("21.5") returns False
    - is_binary_state("idle") returns False
  </behavior>
  <implementation>
    Create `routine_model.py` in `custom_components/behaviour_monitor/` with:

    **Constants:**
    - MIN_SLOT_OBSERVATIONS = 4
    - SLOTS_PER_ENTITY = 168
    - DEFAULT_HISTORY_WINDOW_DAYS = 28
    - BINARY_STATES = {"on", "off", "open", "closed", "locked", "unlocked"}

    **ActivitySlot (dataclass):**
    - event_times: deque(maxlen=56) — stores ISO timestamp strings (not datetime objects, for serialization simplicity)
    - numeric_mean: float = 0.0; numeric_m2: float = 0.0; numeric_count: int = 0 — Welford online accumulators
    - is_sufficient property: True when len(event_times) >= MIN_SLOT_OBSERVATIONS or numeric_count >= MIN_SLOT_OBSERVATIONS
    - record_binary(iso_timestamp: str) — append to event_times
    - record_numeric(value: float) — update Welford accumulators: count += 1, delta = value - mean, mean += delta/count, delta2 = value - mean, m2 += delta * delta2
    - expected_gap_seconds() -> float | None — compute inter-event intervals from consecutive timestamps, return median if sufficient, None otherwise
    - slot_distribution() -> tuple[float, float] | None — return (mean, stdev) if sufficient (stdev = sqrt(m2/count) if count > 1 else 0.0), None otherwise
    - to_dict() / from_dict() — serialize event_times as list of strings, Welford accumulators as floats

    **EntityRoutine (dataclass):**
    - entity_id: str; is_binary: bool; slots: list[ActivitySlot] (168); history_window_days: int = 28; first_observation: str | None = None
    - slot_index(hour, dow) -> int: dow * 24 + hour
    - record(timestamp: datetime, state_value: str) — compute slot_index from timestamp.hour and timestamp.weekday(), then call slot.record_binary or slot.record_numeric; set first_observation if None
    - expected_gap_seconds(hour, dow) -> float | None — delegate to slot
    - daily_activity_rate(target_date: date) -> int — count event_times across all 24 slots for that dow that fall on target_date
    - confidence(now: datetime) -> float — proportional: min(1.0, days_since_first_observation / history_window_days)
    - to_dict() / from_dict()

    **RoutineModel:**
    - __init__(history_window_days=28); _entities: dict[str, EntityRoutine]
    - get_or_create(entity_id, is_binary) -> EntityRoutine
    - record(entity_id, timestamp, state_value, is_binary)
    - overall_confidence() -> float — average of entity confidences, 0.0 if empty
    - learning_status() -> str — "inactive" / "learning" / "ready" based on overall_confidence thresholds
    - to_dict() / from_dict()

    **is_binary_state(state_value: str) -> bool** — module-level function, returns True if state_value.lower() in BINARY_STATES

    CRITICAL: Zero `from homeassistant` imports. All datetime objects passed as parameters. ISO strings used for serialization.
  </implementation>
