    **Add _bootstrap_from_recorder method to coordinator:**

    ```python
    async def _bootstrap_from_recorder(self) -> None:
    ```

    1. Import recorder API: `from homeassistant.components.recorder import get_instance` and `from homeassistant.components.recorder.history import state_changes_during_period`
    2. Calculate start = now - timedelta(days=self._history_window_days)
    3. For each entity_id in self._monitored_entities:
       a. Try: get_instance(self.hass).async_add_executor_job(state_changes_during_period, hass, start, now, entity_id, True, True)
       b. Get states list from result dict
       c. Determine is_binary by checking first non-unavailable state value against is_binary_state()
       d. For each state: skip "unavailable"/"unknown"/None; call self._routine_model.record(entity_id, state.last_changed, state.state, is_binary)
       e. Log debug: "Bootstrap: loaded N states for entity_id"
       f. On exception: log warning, continue
       g. await asyncio.sleep(0) between entities

    **Call _bootstrap_from_recorder in _async_setup:**
