
   ```python
   if not self._track_attributes:
       old_state = event.data.get("old_state")
       if old_state is not None and old_state.state == ns.state:
           return
   ```
   This skips recording when track_attributes=False and the state value hasn't changed (attribute-only change).

Do NOT change any other coordinator logic. The _history_window_days field should remain and continue to be used for bootstrap and stat_training display.
